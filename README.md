# Low Poly City · 数字孪生城市平台

基于 **Three.js** 与 **ECharts** 构建的轻量级、高性能 3D 低多边形城市可视化大屏系统。项目旨在探索纯前端技术栈在数字孪生、智慧城市数据洞察方面的可视化表现力。

## 🌟 核心特性

- **程序化生成城市 (Procedural Generation)**
  动态算力生成庞大城区的区块规划，包含核心 CBD、高级公寓、普通居民区与远郊厂房等数十种多边形建筑类型。
- **极致渲染优化 (Instancing & Geometry Merge)**
  基于 `BufferGeometry` 合批技术压缩静态资源，并利用 `InstancedMesh` 驱动由数百辆穿梭汽车组成的动态车流系统，将海量几何体的极限渲染 Draw Calls 降至百级别以下。
- **动态昼夜交替系统**
  支持无缝衔接的 白天 / 黄昏 / 夜晚 光影主题切换，配合半球光与自定义色系表，夜晚提供炫目的科技感的高楼发光窗格与路网霓虹车灯环境映射。
- **全要素数据大屏解耦挂载**
  利用纯前端 CSS 与事件派发实现 UI 完全解耦，集成南丁格尔图、滑动折线波形心电图、动态渲染引擎算力性能池监控等核心信息图表。
- **Vite 现代化工程架构**
  毫秒级 HMR 本地热更新支持，纯原生 ESM 模块化管理依赖，抛弃冗杂的网络 CDN 阻塞拉取。

## 🚀 快速启动

1. **安装环境依赖**
   请确保本地已拥有 Node.js (含 npm) 运行环境。

   ```bash
   npm install
   ```

2. **启动本地开发热更新服务器**

   ```bash
   npm run dev
   ```

3. 在浏览器中打开命令行输出的本地端口（通常为 `http://localhost:5173`）。

## 🎮 操控指南

- **旋转视角**：鼠标左键按住拖拽
- **缩放镜头**：鼠标滚轮上下滚动
- **平移中心**：鼠标右键按住拖拽
- **纯净全景视图**：点击右上角的“隐藏大屏看板”开关可收起数据大盘，进入沉浸式大世界沙盘鸟瞰模式。

## 🛠️ 技术底座

- **核心 3D 渲染框架**: [Three.js](https://threejs.org/) (v0.170.0+)
- **二维数据可视化大盘**: [ECharts](https://echarts.apache.org/) (v5.5.0+)
- **前端本地化构建驱动**: [Vite](https://vitejs.dev/)
